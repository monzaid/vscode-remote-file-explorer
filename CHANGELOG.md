# Changelog

## 0.1.0 (Initial Release)

- 🌲 Sidebar TreeView for browsing remote directories
- ✏️ Remote file editing via VSCode FileSystemProvider
- 🔌 SSH/SFTP protocol support (ssh2)
- 🔌 FTP/FTPS protocol support (basic-ftp)
- 🔌 Custom Agent protocol support (HTTP/WebSocket)
- 🔍 Remote file search using grep/ripgrep
- 🖥️ SSH terminal integration via Pseudoterminal
- 🔒 Credential encryption via VSCode SecretStorage
- ⚠️ Conflict detection with 3-option resolution
- 📊 Status bar connection indicator
- 🔄 Auto-reconnect with exponential backoff
- 🔐 Permission icons for read-only files
- 🐳 Docker test environment (SSH + FTP)
- 📦 VSIX packaging support
